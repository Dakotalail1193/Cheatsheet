Main Sections of HTML

Tag Name Description
<!DOCTYPE html> Doctype html Document type delcaration. Declaring it is html.
<html> html Added after !DOCTYPE declaration. Should contain all elements and structure.
<head> head Metadata about the page, Title is stored here.
<body> body Main content of the document, body is what is shown on the webpage. Only one per page.
<footer> footer Footer section of document or section, usually contains contant/author information or back to top links.
<title> Title Element What shows on the web tab as the title.

Common Tags and Attributes

Tag Name Description.
<h1>-<h6> Heading Elements Heading elements that lower in size. <h1> being the largest <h6> being the smallest.
<p> Paragraph Element Displays a block of text.
<a> Anchor Element Used to create hyperlinks in HTML document. Other webpages, files on the same server,or other URL using Hyperlink reference (href) attribute.
<div> Div Element Divides document into sections.
<img> Image element Embeds images into document using the <src> attribute.
<video> Video Element Embeds media player into document using <src> attribute. Controls can be added with the <controls> attribute.
<br> Line Break Used to break a line break in text for division.
<em> Emphasis element Puts emphasis on text inside tags, will usually italicize.
<strong> Strong Element Highlights important text in bold by default.
<li> List element Begins an either ordered or unordered list. See below.
<ol> Ordered list Displays list in numerical order appearance
<ul> Unordered List Displays list in unordered appearance using bullets

Table Attributes.

Tag Name Description
<table> Table Wrapper for all tables.
<thead> Table head Header row of table defining the columns in table.
<tbody> Table body Set of rows in table containing actual data.
<tr> Table row Table Row container, contains the <td> element for each row
<td> Table Data Contains the contents in specific cell of the Table Row.
<tfoot> Table Foot Rows set for the footer of table.
